# Analise-_Dados_Hospital
Obejtivo Principal: Analisar de dados para resolução de gargalos no controle  e acompanhamento de leitos do Hospital Vida Plena.


Problemas encontrados nos dados:
 - Leitos ficam bloqueados por muito tempo
 - Podem ser reutilizados sem limpeza adequada
 - Leitos ficam vazios por falta de aviso
 - Perda financeira
 - Falta de integração entre areas 


O que fazer:

- Monitorar a Ocupação e disponibilidade dos leitos
- Facilitar a comunicação entre equipes (Enfermagem, Higienização e Administração)
- Gerar relatorios de ocupação e tempo médio de permanencia
- Sinalizar a quantidade de leitos impactados por manutenção
- Propor indicadores para acompanhamento da eficiência do processo

Metas:

- Tranformar o processo manual em algo automatizado, rastreavel e transparente
- Minimizar falhas manuais e propor o mapeamento visual do fluxo dos leitos

  Processos e tecnologias:

  - Base de dados (Excel/CSV) - (Criar e estruturar os dados, Simular os dados de leitos, pacientes em movimentação)
  - Analise e tratamento (Python) - (Detectar e tratar inconsistencias, Analisar os dados e gerar indicadores)
  - Visualização de dados (Power BI) - (Ler e interpretar os dados, Criar objetos visuais, Storytelling dos dados)
